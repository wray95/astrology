# 20-Celebrity Shrinkhala (Srinkhala) Research — Composite Re-Rank

**Method (user link):** https://www.jyotishvidya.com/srinkhala.htm
- **Srinkhala** = a *closed* chain of **3+ planets** linked by mutual dispositors (A in B's sign, B in C's sign, C in A's sign…). It is the "younger sister of Parivartana" (2 planets).
- **Strength rule (critical):** Parivartana bond = 100; **3-planet Srinkhala = 50; 4-planet = 33; 5-planet = 25**. *"It is best to consider only a 3-planet Srinkhala… ignore the same if there are more planets involved as the effect… is not considerable."* → **only the 3-loop is meaningful; 4/5-loop is weak/ignore.**
- **Interpretive rule:** planets in a Srinkhala *"behave as if they are placed in their own signs"* (gain strength). Example given: Roger Federer's 3-loop (Sun–Moon–Venus) → tennis champion.
- This **corrects** the earlier Reddit-based reading (Turns 5–6) that celebrated P3's 5-loop as "strongest" — per this link, a 5-loop is weak and should be ignored.

**Data sourcing (link-only, per standing rule):** birth date/place + bio (career/wealth/foreign/marriage) from web (Wikipedia/biography links); **planetary positions from Drik Panchang** (`planetary-positions-sidereal`, **Lahiri/Chitra Paksha** ayanamsa — confirmed on the page). Planet *signs* are date+time based (location-independent); Lagna is location-dependent so not used for loop detection. Celebrities without a known birth **time** use 12:00 and have Moon flagged approximate.

**Composite axes (same framework as report §12):** career · wealth · foreign · success · sudden gains · marriage.
